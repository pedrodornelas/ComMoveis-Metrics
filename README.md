# Análise de Métricas de Desempenho

Projeto de modelagem de canais Rayleigh e Rice com foco na análise de desempenho do sistema utilizando as métricas de probabillidade de interrupção e probabilidade de erro de símbolo.

Projeto gerado para a disciplina de Comunicações Móveis do Programa de Pós Graduação em Engenharia Elétrica da Universidade de Brasília (UnB).
